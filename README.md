# Academix Database

Academix Database is a relational database designed for a Student Academic & Mentorship Management System. The project demonstrates database design, normalization, table relationships, and SQL implementation using MySQL.

## Technologies
- MySQL
- SQL

## Features
- 9 Relational Tables
- Primary & Foreign Keys
- Normalized Database Design
- Sample Data
- User Authentication Table
- Attendance Management
- Assessment Management
- Faculty & Student Management
- Subject & Course Management

## Database Tables
- College
- Course
- Department
- Faculty
- Student
- Subject
- Attendance
- Assessment
- Users

## Future Scope
This database will serve as the backend for the **Academix Student Management System**, which will be developed using Spring Boot, Thymeleaf, Bootstrap, and MySQL.
